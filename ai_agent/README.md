# Coin Identifier Landing Page

This directory contains a static landing page for the app. It uses the same visual language as the Flutter project:

- Deep burgundy background: `#39070A`
- Muted red surfaces: `#7B1717` and `#5F1212`
- Aged gold borders and secondary text: `#D0AA6B`
- Antique silver primary CTA: `#C0C0C0`
- Playfair Display for titles and Montserrat for body copy

Open `index.html` in a browser to view it. The page references existing project imagery from `../images`, so no app files or shared assets were modified.
